## how to run project
- 1. cd manga_hub
- 2. Terminal: go build -o mangahub main.go (delete old mangahub.exe if exists)
- 3. rename mangahub -> mangahub.exe
- 4. add "D:\Github Repository\Go_MangaHub\manga_hub" to your system environment variables Path
- 5. Terminal: mangahub server start (this runs the HTTP port 8080)
